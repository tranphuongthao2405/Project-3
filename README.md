### Guide line

- Download node_modules in client (cd client) and in root folder for server :

```bash
npm i
```

- Run in dev zone:

```bash
npm run dev / npm run backend
```

- Run frontend:

```bash
cd frontend: npm run start --prefix client
```

- Run backend:

```bash
nodemon server/index.js
```

- Start demo at localhost:3000:
